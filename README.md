## როგორ ჩავრთოთ ლოკალური ვერსია?
- უნდა გვქონდეს ruby-2.x.x
- დავაყენოთ bundler (`gem install bundler`)
- დავაყენოთ დამოკიდებულებები (`bundle install`)
- ჩავრთოთ ლოკალური ვერსია (`bash serv`)
